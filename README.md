# SIM-000 Core System

Статус: active

Мета
- Задати базову «фізику» Simulation v2.
- Визначити структуру репозиторію (`/control`, `/sim`, інші ядрові папки).
- Описати формат симуляцій (SIM-XXX) та їхні статуси.

Область відповідальності
- Головні правила системи (що вважається реальністю, як фіксуємо рішення).
- Структура директорій та ідентифікаторів симів.
- Базові угоди по гілках (`main`, `sim/xxx`, тощо).

Артефакти
- `control/PHYSICS.md` — формалізовані правила системи.
- `control/REGISTRY.md` — реєстр усіх симуляцій (SIM-000, SIM-001, SIM-002, ...).
- Описана структура папок `/sim/` з прикладом хоча б однієї симуляції.

Критерії завершення
- Створено й заповнено `control/PHYSICS.md`.
- Створено й заповнено `control/REGISTRY.md` з мінімум 1–3 симами.
- Для активних SIM створені папки в `/sim/` з власними `README.md`.
- Опис узгоджений з фактичною структурою репозиторію.
